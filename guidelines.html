Typography

It is extremely important to keep typography consistent
on all digital and printed assets.

Proxima Nova

We use exclusively the "Proxima Nova" font family for both digital and printed assets. Proxima Nova is a very flexible and big font family with a lot of different styles available.

Usage

Web

We have the web font version ready to be embedded, so if you are working on a page under the *clevertech.biz domain you can use Proxima Nova by including this code on the HEAD of your code. Once included check the headlines and paragraphs & lists sections for style references.

If you are working on a different domain you can add it on the Typekit control panel. Find the credentials here.

Design Apps

If you want to use Proxima Nova on your favorite design app you need to get a copy of the font and install it locally. If you don't own the font please contact Adrian Salazar on the Design channel.

Bootstrap

We have a Bootstrap theme fully loaded with the
Clevertech branding in this guide. Here's a preview!

Usage

  • Download a copy of bootstrap (not source) and replace reference to bootstrap-theme.css with theme.css in html. Put theme.css in /css folder.
  • Download Icon Fonts and place on the root of your project in a folder called /fonts this theme uses the solid icon pack only, download pack from Icons section for class references
  • Embed brand font with typekit (if not *.clevertech.biz). See Typography section for more information.
  • You're all done! All info regarding components and general bootstrap usage can be found on Bootstrap docs.
